# Mali - Twitter Clone

Backend API using Nest Framework (NodeJS + TypeScript + PostgreSQL )

![NodeJS Workflow](https://github.com/<OWNER>/<REPOSITORY>/actions/workflows/nodejs.yml/badge.svg)

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
